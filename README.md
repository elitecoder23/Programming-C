## Anish Nag - Learning how to program in C ##

## Description ##

I am teaching myself how to program in C. This idea came from watching Dr. Jonas Birch on youtube.
His video, "How to become fluent in ANY programming language in 6 weeks" was intriguing. Therefore, I decided to take on this challenge.

Rules of the challenges:
  - Cannot google for solutions
  - Cannot use any AI tool or Large Language Model
  - Only read official documentation(I added this rule because to learn C, I need to read the documentation on their website obviously)


The human brain is very smart. It is wired to learn languages quickly, including programming languages.

The idea is to program everyday. I will only read official documentation. If I get stuck, I will just have to figure it out. 
This will force my brain to work hard and learn the language.
